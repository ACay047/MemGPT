# [MemGPT](https://memgpt.ai)


## Setup

Set up dependencies:

```sh
pip install -r requirements.txt
```

Add your OpenAI API key to your environment:

```sh
export OPENAI_API_KEY=YOUR_API_KEY
```

## MemGPT CLI

To run MemGPT in CLI mode, simply run `main.py`:

```sh
python3 main.py
```
